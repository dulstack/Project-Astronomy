# Project Astronomy
Port of Celestial to Automated Linux From Scratch
## Building
------------------------------
These packages are needed to build the project: `meson fakeroot cpio ninja gperf bison flex diffutils libelf-dev wget make gcc` and python package `jinja2`. The required packages to build the compiler and binutils can be found in [https://wiki.osdev.org/GCC\_Cross-Compiler](https://wiki.osdev.org/GCC_Cross-Compiler)

To build, clone this repository and run `make`.
